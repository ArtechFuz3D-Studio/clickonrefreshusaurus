# Tables: TR, TD and TH Elements

```
<table>
<td>
<tr>
<th>
<thead>
<tbody>
<tfoot>
<colgroup>
<caption>
```

tr = table row
td = table data
th = table header ( different to table head, thead)

Make a table using these elements displaying the heaviest living bird species.
